ActionCode intigiriert einen Menüpunkt zur Administration und ein Widget zur Anzeige, organisiert Codwörter für Weiterleitungen

== Description ==

ActionCode etabliert eine vollständige Aktionscode Funktion, damit kann man evtl. als Projekt eine Website haben, und die zip kann mann dann
durch Eingabe im Widget downlaoden z.B. das Projekt Meine erste Homepage: projekt_meh

== Installation ==

1. Lade den Ordner "wp_plugin_actioncode" hoch, in den Ordner "/wordpress/wp-content/plugins"
2. Ändere die Datenbankdaten in der "conf.inc.php", zusätzlich wirst du immer aufgefordert die Konfigurationsdatei zu ändern!
3. Führe die install.php aus. Bequem über den Link in der "ActionCode Property" Seite
4. Öffne die "ActionCode Property" Seite unter der Rubrik "Plugins" und du kannst die ActionCodes einsehen
5. Lösche, verändere und erstelle neue ActionCodes durch manuelles Einfügen von Datensätzen in der Datenbank, sehr einfach mit phpMyadmin
